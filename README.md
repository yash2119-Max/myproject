# Voice Emotion Detection Using Bi-GRU

## Project Overview
This project detects human emotions from speech audio using Deep Learning techniques.

## Dataset
- RAVDESS Dataset
- 2880 audio samples

## Feature Extraction
- MFCC (Mel Frequency Cepstral Coefficients)

## Model
- Bidirectional GRU (Bi-GRU)

## Technologies Used
- Python
- TensorFlow
- Keras
- Librosa
- NumPy
- Scikit-Learn

## Results
- Test Accuracy: 80.2%

## Emotions Detected
- Happy
- Sad
- Angry
- Fearful
- Calm
- Neutral
- Disgust
- Surprised
